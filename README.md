# DecodeLabs-Internship
Password Strength Checker

Project Overview

This project checks the strength of a password based on different security criteria.

The program analyzes:

- Password length
- Uppercase letters
- Lowercase letters
- Numbers
- Special characters

Based on these checks, the password is classified as:

- Weak
- Medium
- Strong

Features

- User-friendly input
- Password strength analysis
- Simple and easy-to-understand code
- Beginner cybersecurity project

Technologies Used

- Python

How to Run

1. Download the project files.
2. Open a terminal or command prompt.
3. Run the following command:

python password_strength_checker.py

4. Enter a password when prompted.
5. View the password strength result.

Example

Input:
Password@123

Output:
Strong Password
